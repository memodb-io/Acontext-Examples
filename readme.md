## `acontext-cli` examples

Supported Examples:

- `python/acontext_basic`: basic sdk examples.
- `python/agno_basic`: use agno and save the data to acontext.
- `python/openai-agent-basic`: use openai agent sdk and save the data to acontext.



### Install cli

```bash
curl -fsSL https://install.acontext.io | sh
```



### Download templates

```bash
acontext create my-project --template-path "python/agno_basic"
```
